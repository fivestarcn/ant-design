通用按钮。

## 何时使用

- 通用按钮。

## API

| 参数      | 说明                                     | 类型          | 默认值 |
|-----------|------------------------------------------|---------------|--------|
| disableBefore   | 不可选择的日期（开始） | function | 无  |
| disableAfter   | 不可选择的日期（结束）  | function | 无  |
| label   | 定义输入框名称，当label未赋值时默认只有date输入框,隐去label标签 | string | 无  |
