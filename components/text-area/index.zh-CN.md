多行文本输入框。

## 何时使用

- 用户需要输入的文字较多时。

## API

| 参数      | 说明                                     | 类型          | 默认值 |
|-----------|------------------------------------------|---------------|--------|
| value   | 背景色   | string        | 无  |
| label   | 标签 | string        | 无  |
| onChange   | 输入值发生变化时的回调 | function        | 无  |
| hidden   | 是否显示文本输入框 | bool       | false  |
