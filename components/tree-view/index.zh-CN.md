## 何时使用

## API

| 参数      | 说明                                     | 类型          | 默认值 | 必须参数 |
|-----------|------------------------------------------|---------------|--------|---------|
| checkable   | 是否选中  | bool        | 无  |      |
| onToggle   | 收缩或展开操作界面 | function        | 无  |   yes   |
| onChange   | 输入值发生变化时的回调 | function        | 无  |      |
| data   | 数据 | array,object       | 无 |      |
