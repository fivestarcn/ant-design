弹出树组件的input。

## 何时使用

- 用户需要弹出树组件。

## API

| 参数      | 说明                                     | 类型          | 默认值 |
|-----------|------------------------------------------|---------------|--------|
| children   | 需要操作的节点   | element        | 无  |
| label   | 标签 | string        | 无  |
| onChange   | 输入值发生变化时的回调 | function        | 无  |
| show   | 是否显示弹出树 | bool       | false  |
| data   | 数据 | array,object       | 无 |
